# 🎯 Logic Box - Menu Driven Python Program

## 📖 Overview
A simple menu-driven Python program that demonstrates:
- ⭐ Pattern Generation
- 🔢 Number Analysis (Odd/Even)
- ➕ Sum Calculation
- 🚪 Exit Option

---

## ✨ Features

### 1️⃣ Generate Pattern
Prints a star pattern based on the number of rows entered by the user.

### 2️⃣ Analyze Numbers
- Checks whether numbers are Odd or Even.
- Calculates the total sum of the range.

### 3️⃣ Exit
Closes the program.
